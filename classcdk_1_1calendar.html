time_t cdk::calendar::activate ( chtype *  actions)
inline

activates the calendar widget and lets the user interact with the widget.

Parameters
actionsIf the actions parameter is passed with a non-NULL value, the characters in the array will be injected into the widget. To activate the widget interactively pass in a NULL pointer for actions.
Returns
If the character entered into this widget is RETURN then this function will return a type of time_t. If the character entered into this widget was ESCAPE or TAB then this function will return a value of (time_t)-1 and the widget data exitType will be set to vESCAPE_HIT.
